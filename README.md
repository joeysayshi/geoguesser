# GeoGuesser (Mimic + Home)

Now includes a dedicated home page inspired by the provided layout.

## Pages

- `index.html` -> redirects to `home.html` if signed in, otherwise `login.html`
- `login.html` -> sign in/register
- `home.html` -> hub page with top nav, hero, large cards, and play entry points
- `game.html` -> Street View gameplay screen

## Home features

- Dark top navigation bar with category links
- Large centered title/hero text
- Three large mode cards styled like the reference
- Dot indicator row and right-side utility rail
- Buttons wired to gameplay (`game.html`), including daily mode
